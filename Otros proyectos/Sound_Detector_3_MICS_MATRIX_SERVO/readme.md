# Localizador de sonido 
El siguiente proyecto esta basado en el instructable (http://www.instructables.com/id/Sound-Locator/), he modificado el codigo y agregado comentarios en español, los materiales que necesitas son 3 microfonos, 1 servo motor, 1 matriz 8x8 max232 y un arduino uno.
